Binary Polynomial Regression Formula
y = β₀ + β₁·x₁ + β₂·x₂ + β₃·x₁² + β₄·x₁·x₂ + β₅·x₂² + … + βₙ·x₁ᵐ·x₂ⁿ + ε
y = session score (0–100) x₁ = fluid volume (ml) x₂ = exchange time (min) ε = error term
Scoring Rules
◆
Perfect Session
20 minutes, 2000ml → 100 points
◆
Time Deviation
−2.5 points per minute off target (±20 min)
◆
Volume Deviation
−2.5 points per 100ml off target (±2000ml)
◆
Catheter Displacement
Alert if drainage is 200–500ml less than infusion
◆
Air Embolism Risk
Alert if venting step not completed
Example Calculation
18 min, 2100ml → |20−18|=2min, |2000−2100|=100ml
Score = 100 − (2.5×2) − (2.5×1) = 92.5 pts
